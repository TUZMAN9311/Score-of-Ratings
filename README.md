# Score of Ratings 📊

Простой, быстрый и адаптивный кроссплатформенный калькулятор среднего балла, разработанный на базе **.NET 10 (Preview)** и **.NET MAUI**. 

Приложение создано для школьников и студентов, которым нужно быстро прикинуть, как изменятся их четвертные или семестровые оценки, если они получат несколько новых баллов.

## ✨ Особенности
* **Умный расчет:** калькулятор не просто считает среднее арифметическое, но и сразу показывает итоговую округленную оценку, которая пойдет в аттестат.
* **Адаптивный интерфейс:** благодаря оптимизированной верстке с использованием `ScrollView`, интерфейс идеально подстраивается под любые экраны — от старых бюджетников (вроде Redmi 9A на Android 11) до самых современных флагманов на Android 13/16.
* **Валидация данных:** встроенная защита от некорректного ввода (приложение не упадет, если случайно ввести буквы вместо цифр).
* **Многостраничная навигация:** удобное боковое меню с вкладками «Главная» и «Контакты» для связи с разработчиком.

## 🛠 Стек технологий
* **Язык программирования:** C#
* **Фреймворк:** .NET MAUI (.NET 10.0)
* **Разметка:** XAML (с включенной генерацией исходного кода `SourceGen` для максимальной производительности)

## 🚀 Сборка и установка

Если вы хотите собрать приложение самостоятельно из исходников, вам понадобится установленный .NET 10 SDK с рабочей нагрузкой (workload) `maui-android`.

### Универсальная сборка под все архитектуры (ARM32 / ARM64):
Чтобы собрать один универсальный `.apk` файл, который запустится и на старых 32-битных процессорах, и на современных 64-битных устройствах, выполните в терминале:

```bash
dotnet publish -c Release -f net10.0-android /p:AndroidCreatePackagePerAbi=false

После завершения сборки готовый пакет будет лежать по пути:
bin/Release/net10.0-android/
